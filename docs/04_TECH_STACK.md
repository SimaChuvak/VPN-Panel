# Технологический стек VPN Panel

## Назначение

Документ определяет официальный технологический стек проекта.

Использование других технологий допускается только после обсуждения и обновления документации.

---

# Backend

## Язык

Python 3.13

Причины выбора:

- современный синтаксис;
- высокая производительность;
- отличная поддержка async;
- долгосрочная поддержка.

---

## Web Framework

FastAPI

Используется для:

- REST API;
- OpenAPI;
- Dependency Injection;
- валидации запросов;
- асинхронной обработки.

---

## ASGI Server

Uvicorn

Назначение:

- запуск FastAPI;
- обработка HTTP-запросов;
- поддержка async.

---

## ORM

SQLAlchemy 2.x

Используется исключительно в современном стиле SQLAlchemy 2.

Запрещается использовать устаревший API.

---

## Миграции

Alembic

Используется для:

- создания миграций;
- обновления схемы БД;
- контроля структуры базы данных.

---

## Валидация

Pydantic v2

Используется для:

- Request Models;
- Response Models;
- DTO;
- конфигурации приложения.

---

## PostgreSQL Driver

asyncpg

Используется только асинхронное подключение.

---

## Конфигурация

pydantic-settings

Используется для загрузки:

- .env
- ENV Variables
- Production Config

---

# Принципы выбора технологий

При выборе новых библиотек учитываются:

- активная поддержка;
- совместимость с текущим стеком;
- производительность;
- безопасность;
- долгосрочная перспектива;
- качество документации;
- размер сообщества.

Новые зависимости должны быть обоснованы и задокументированы.
